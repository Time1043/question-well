# yudada
- Introduction

  yudada 是一款基于 Vue3 + SpringBoot + Redis + ChatGLM + RxJava + SSE 的AI答题应用平台。

  用户可以基于 AI 快速制作并发布答题应用，支持检索、分享、在线答题并基于 AI 得到回答总结；管理员可以集中管理和审核应用。
